# Solar Flare Monitoring & IoT Alert System

## 📌 Description
This project monitors solar flare data using satellite information and provides real-time alerts using an IoT device.

## 🚀 Features
- Real-time solar flare monitoring
- IoT-based alert system using ESP8266
- LED status indication

## 🔧 Technologies Used
- Python
- ESP8266 (NodeMCU)
- Arduino IDE

## ⚙️ Working
1. Fetch solar flare data from NOAA
2. Analyze intensity
3. Trigger IoT alert if threshold exceeded
